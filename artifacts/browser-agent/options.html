Enterprise Tamper-Proof Deployment

To prevent users from disabling or removing this extension, deploy via managed browser policy. When deployed this way, the extension shows 🔒 Managed and users cannot remove it.

Chrome / Edge
ExtensionInstallForcelist via Google Workspace Admin, Intune, or ADMX Group Policy
Firefox
force_installed in Mozilla enterprise policies.json or via Intune
Safari / iOS
Deploy via Apple MDM (.mobileconfig profile). Supervised devices only.
Android
Chrome extensions not supported. Use AIGO-X Native Agent or managed Chrome kiosk mode.

Policy templates are in deployment/ in the extension package. Download from AIGO-X → Agents → Browser Extension → Download Policies.
